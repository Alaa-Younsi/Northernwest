# Northernwest Custom Shopify Theme

## Overview
This is a fully custom Shopify Online Store 2.0 theme for Northernwest, engineered as a premium, minimalist gaming accessories storefront. The design is intentionally bold, high contrast, and product first, while staying fast, clean, and conversion focused.

## Author
Built by Alaa Younsi, professional web developer and creator of this custom Northernwest theme.

## Screenshots


## Design Direction
- Visual language: black, white, and red only, with sharp typography and controlled spacing.
- Brand fit: a premium, performance focused gaming aesthetic with a modern, industrial tone.
- UX goals: fast scanning, clear hierarchy, and strong product emphasis.

## Tech Stack
- Shopify Liquid
- Shopify Online Store 2.0 JSON templates
- HTML
- CSS (custom utility + component system)
- Vanilla JavaScript

## Site Architecture
- layout/: Base layout, head metadata, global assets.
- templates/: JSON templates for product, collection, cart, pages, and list collections.
- sections/: Modular blocks for hero, visuals, categories, best sellers, value strip, product page, cart drawer, header, footer.
- snippets/: Reusable UI components like product cards and icons.
- assets/: Global CSS and JS.
- config/: Theme settings.
- locales/: Translation strings.

## Key Features
- Custom product page layout with gallery, variants, trust badges, and structured content sections.
- AJAX cart drawer with live quantity updates and removals.
- Responsive grid system and mobile optimized layout.
- Custom collection listing with image cards and hover states.
- Variant to image syncing for accurate product visuals.
- Custom SVG visuals for hero, keyboard, and headphones sections.

## Performance & Best Practices
- Minimal dependencies with lightweight vanilla JS.
- Deferred scripts and lazy loaded media.
- Modular section based architecture for easy maintenance.
- Clean, consistent CSS utilities to avoid style drift.

## SEO & Accessibility
- Canonical URL and meta description support.
- JSON LD structured data for products.
- Accessible labels, semantic HTML, and alt attributes.

## Security & Platform Alignment
- Shopify native cart and checkout only.
- No custom checkout logic or sensitive data handling.
- Theme built to align with Shopify best practices.
